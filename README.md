# Job Data Web Scraping & Analysis Project

## 📖 Project Description
This project is a basic Machine Learning and Web Scraping project in which job-related data is collected from a job website and then analyzed.

The scraper extracts the following information:
- Company Name  
- Job Title  
- Job Location  
- Salary  
- Job Description  
- Experience Required  

The collected data is stored in a CSV file and can be used for data analysis or machine learning models.

---

## 🎯 Project Objective
The main goal of this project is:
- To automatically collect job data from a website  
- To clean and organize the scraped data  
- To analyze job trends like salary, company demand, and job titles  
- To prepare data for machine learning models  

---

## 🛠 Technologies Used
- Python  
- BeautifulSoup / Scrapy (for web scraping)  
- Pandas (for data handling)  
- NumPy  
- Matplotlib / Seaborn (for visualization)  
- Scikit-learn (for ML model, if used)

---

## 📊 Features
- Scrapes job data from website  
- Extracts:
  - Company Name  
  - Job Title  
  - Salary  
  - Location  
  - Job Description  
- Saves data in CSV format  
- Performs basic data analysis  
- Can be extended for ML prediction  

---

## 📁 Dataset
The dataset is generated by scraping job listings from a job website.  
Columns in dataset:
- Company Name  
- Job Title  
- Salary  
- Location  
- Description
